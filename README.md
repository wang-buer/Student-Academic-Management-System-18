# 学生教务管理系统18



## 1、项目介绍

学生教务管理系统18只有管理员角色，具有专业管理、学院管理、课程管理、学生管理等功能，适合课程作业的学习

## 2、项目技术

后端框架：Spring

前端技术：jsp

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：Tomcat 7-10版本均可
- maven版本：无限制
- 硬件环境：Windows
